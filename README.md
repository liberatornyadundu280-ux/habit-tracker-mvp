# Habit Tracker (MVP)

## Project Goal
This project is built by students to practice full-stack development, teamwork, and real-world workflows.

The focus is on:
- Planning
- Collaboration
- Deadlines
- Integration

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB
- Version Control: Git & GitHub

## MVP Features
- Add a habit
- View habits
- Mark habit as completed
